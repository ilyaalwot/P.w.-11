# Практична робота №11
## Словники. Map та HashMap в Java Collections Framework

У даній практичній роботі реалізовано систему користувачів на основі практичної роботи №10, але замість множини HashSet використано словник HashMap.

## Реалізовано:
- клас UserIdentifier, який використовується як ключ у HashMap;
- клас User, що містить UserIdentifier, пароль, дату останнього входу та стан авторизації;
- клас UserRegistry для роботи з користувачами;
- реєстрацію користувача;
- вхід у систему;
- вихід із системи;
- видалення користувача;
- перевірку реєстрації користувача;
- отримання списку користувачів у вигляді LinkedList;
- сортування користувачів через лямбда-вираз;
- фільтрацію користувачів через Predicate;
- консольне меню для демонстрації роботи програми.

## Використані технології
- Java
- Map
- HashMap
- LinkedList
- Comparator
- Predicate
- Lambda expressions
